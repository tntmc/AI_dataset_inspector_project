## Cara menggunakan fitur di Lyzer:
- Buka web Lyzer klik browse file pada sidebar menu file upload.
- Cari file yang ingin dianalisis (excel, json, csv) dan klik open
- tunggu hingga web memproses hasil dataset dan insightnya sebelum menggunakan chatbot
- Khusus excel apabila memiliki lebih dari 1 sheet, pililah pada menu sheet pada bagian atas dashboard setelah analisis
